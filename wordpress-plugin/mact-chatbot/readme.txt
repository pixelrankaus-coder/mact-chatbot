AI chatbot widget for your website.

== Description ==

Adds the MACt AI chatbot to your WordPress site.

== Installation ==

1. Upload the `mact-chatbot` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Go to Settings → MACt Chatbot
4. Enter your Widget URL
5. Check "Enable Chatbot"
6. Save
